# 中美关税博弈交互式仿真系统（课程大作业）

简短说明  
本项目为博弈论课程大作业，基于 2025 年相关贸易与宏观经济数据，提供一个可交互的前端仿真与可视化平台，用于演示静态与动态博弈、混合策略与情景分析，并集成简单的 AI 策略推荐模块。

主要目标  
-用可视化与交互方式帮助理解囚徒困境、动态博弈树、混合策略均衡与策略敏感性。  
-支持参数化实验（收益、权重、概率、关税等）以观察结果变化。  
-提供可复现的教学演示与实验平台，便于课堂汇报与报告撰写。

功能一览（详尽）
-系统概览
  -项目简介、关键经济指标与快速入门提示。
  -展示 2025 年关税峰值、GDP 与股市冲击等实证信息。

-静态博弈（囚徒困境）
  -可交互调整收益矩阵（R/T/P/S）并实时计算 payoff。
  -可调整 GDP / 贸易 / 股市权重（会按偏差缩放 payoff，能明显观察滑块影响）。
  -自动检测纯策略纳什均衡并高亮矩阵单元。
  -提供理论背景说明与基于真实数据的解释。

-动态博弈仿真（决策树）
  -可展开/折叠的博弈树，支持剪枝（自动移除不现实分支）与节点着色（对抗/合作/最优）。
  -叶节点显示基于当前参数的动态 payoff（悬停可查看完整描述）。
  -支持漫游、拖拽、缩放；图例置于右上角。
  -后向归纳分析与最优路径标注。

-混合策略与敏感性分析
  -允许设置双方合作概率 p、q，计算期望收益与理论混合均衡。
  -敏感性分析图展示合作概率变动对期望收益的影响。

-情景模拟器
  -预设三类情景（升级期 / 降温期 / 缓冲期）。
  -支持自定义参数（关税、持续时间、合作可能性），输出综合风险评分与建议。
  -缓冲期提供长期影响预测图。

-AI 策略推荐（简单模型）
  -基于当前关税、政治压力、经济增长、技术依赖等指标生成策略建议（带置信度、理由与行动建议）。
  -同步展示 AI 指标条形图，方便解释决策依据。

可视化与交互细节
-ECharts 渲染所有图表（树图、折线、柱状图）。  
-决策树采用水平布局以减少文字重叠，并对节点文本做截断显示，完整信息在 tooltip 中可见。  
-节点颜色：对抗（红）、合作（绿）、最优（黄）；并在图上显示图例。  
-剪枝规则（默认）：当当前关税 > 80 或 自定义合作可能性 < 30% 时会自动移除“主动合作/直接合作”分支（可定制）。

安装与本地运行（Windows）
1.最简便（用于提交/演示，需联网）：在文件管理器中双击或右键选择浏览器打开 `index.html`（Chrome/Edge/Firefox）。  
   -要求：能访问互联网（项目依赖 Vue、ECharts、Font Awesome 的 CDN）。  
   -说明：若浏览器对本地文件的某些请求有限制（少见），改用下面的本地服务器方法即可。

2.推荐（开发/稳定）：使用 VS Code + Live Server 插件。  
   -打开项目文件夹，右键 `index.html` → Open with Live Server（自动在默认浏览器以 [http://127.0.0.1:5500/] 打开）。

3.轻量本地静态服务器（命令行，项目根目录）：  
   -Windows PowerShell / CMD（需安装 Python）：
     python -m http.server 8000
     然后在浏览器访问： [http://localhost:8000/]

4.说明：打开后使用左侧导航切换模块并调整参数。如遇图表不显示或报错，请查看浏览器控制台（F12）；推荐先尝试第1步，若有跨域或本地资源限制再切到第2或第3步。

关键文件说明
-index.html — 页面结构与模块容器（引入 Vue、ECharts、样式与脚本）。  
-style.css — 全局样式、布局与颜色主题。  
-script.js — Vue 实例、交互逻辑、所有图表渲染（包括决策树剪枝/着色/tooltip 等）和 AI 推荐逻辑。  
-README.md — 本文档（项目说明与使用指南）。

可配置项（开发者注意）
-剪枝规则：在 script.js 的 renderGameTreeChart() 中调整 pruneCooperation 的判断逻辑以适配不同情景。  
-payoff 模型：calculatePayoff() 中权重影响系数可调（当前约 6%/单位偏差），可替换为更复杂的加权分项模型。  
-决策树样式：在 renderGameTreeChart() 中修改 `orient/levelGap/nodeGap/label.formatter` 以改善布局与文本显示。

调试与常见问题
-若图表为空或控制台报错：按 F12 查看错误；常见原因是 ECharts 或 Vue CDN 无法加载，请确保网络或改为本地依赖。  
-若树图文字仍重叠：可在 script.js 中增大 `levelGap` 与 `nodeGap`，或设置 `initialTreeDepth` 较小以折叠深层节点。  
-权重滑块无明显效果：已实现为“偏差缩放”逻辑，若需线性分项影响请告知具体公式。
